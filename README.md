# Contact Book

## Overview

Contact Book is a beginner-friendly Python project that helps users store and manage contact information. It allows adding, viewing, searching, and deleting contacts using dictionaries.

## Features

- Add new contacts
- View all contacts
- Search contacts by name
- Delete contacts
- Menu-driven interface

## Technologies Used

- Python

## Concepts Covered

- Dictionaries
- Loops
- Conditional Statements
- User Input
- Dictionary Methods
